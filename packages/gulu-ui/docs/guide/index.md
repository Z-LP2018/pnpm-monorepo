# 指南

欢迎使用 **@gulu/ui** 组件库！这是一个基于 Vue 3 构建的现代化 UI 组件库，提供了丰富的组件和良好的开发体验。

## 环境要求

在使用本组件库之前，请确保你的开发环境满足以下要求：

- **Node.js**: >= 20.0
- **Vue**: >= 3.5
- **TypeScript**: >= 5.0（推荐）

## 特性

- 🎨 **现代化设计** - 基于 Tailwind CSS，提供美观的 UI 组件
- 📦 **TypeScript 支持** - 完整的类型定义，提供更好的开发体验
- 🚀 **按需引入** - 支持按需引入，减小打包体积
- 🔧 **易于使用** - 简洁的 API 设计，快速上手
- 📱 **响应式设计** - 适配各种屏幕尺寸
